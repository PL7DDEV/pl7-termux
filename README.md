# 🚀 PL7 TERMUX

> Injetor de arquivos automático para Termux com interface bonita e colorida.

![Status](https://img.shields.io/badge/status-online-brightgreen)
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Platform](https://img.shields.io/badge/platform-Termux-purple)

## 📥 Instalação

Cole esse comando no Termux:

```bash
curl -sL https://raw.githubusercontent.com/PL7DDEV/pl7-termux/main/install.sh | bash
```

## 🚀 Como usar

Depois de instalar, digite:

```bash
pl7
```

## 🎯 Recursos

- 💎 Injeção de Holograma
- 🎯 Injeção de HS
- ⚡ Injeção de FPS Boost
- 🔥 Modo All-in-One (injeta tudo)

## ⚙️ Configuração

Edite o caminho de injeção no arquivo `pl7.py`:

```python
INJECTION_PATH = "/sdcard/Download/PL7"
```
